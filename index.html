The stack

Node.js server side:

  • CoffeeScript
  • Express
  • Mocha
  • Sinon
  • Underscore

Stasis statically generates client side files:

  • Bootstrap
  • Haml, Sass, CoffeeScript, and many other languages
  • jQuery
  • Modernizr
  • SmartAsset
  • Underscore

Process management:

  • Foreman
  • Supervisor

Install these

  • RubyGems
  • NPM
  • CoffeeScript - npm install coffee-script

Start a new project

git clone git@github.com:winton/node_template.git
cd node_template
cake new

Follow the instructions at the end.
